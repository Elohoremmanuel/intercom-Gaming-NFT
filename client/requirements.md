## Packages
framer-motion | Page transitions and scroll-triggered animations
lucide-react | High-quality icons for features and UI elements
react-hook-form | Form state management for the contact form
@hookform/resolvers | Zod validation resolver for forms
clsx | Conditional Tailwind class composition
tailwind-merge | Safe merging of Tailwind utility classes

## Notes
- Theme is strictly dark mode to fit the on-chain gaming/tech aesthetic.
- The platform is a plug-and-play middleware SDK for game developers.
- Custom fonts (Outfit for display, DM Sans for body) need to be imported in index.css.
- Using wouter for routing as requested.
